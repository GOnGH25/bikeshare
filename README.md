# Bikeshare project
The project aims to give you some **statistics of bikesharing services** like e.g. _most common month_, _most common day_ or _most common hour_ in different cities. You can find sample data e.g. for NYC [here](https://citibikenyc.com/system-data).

### Date created
This project was created on 2025-12-10.

### Project Title
I call the project simply bikeshare statistics.

### Description
As mentioned briefly in the intro this project aims to give some statistics for bikesharing services in different cities (see [Files used](#files-used)). It is possible to filter by city, month and weekday. The statistical data that are calculated are:
* time statistics
    * most common month
    * most common day
    * most common hour
* station statistics
    * most common start station
    * most commond end station
    * most common trip (start station -> end station)
* trip duration statistics
    * total travel time
    * mean travel time
* user statistics
    * gender
    * user type
    * birth year

### Files used
The `bikeshare.py`script uses the following trip data files (not included in the repo for file size reasons):
* chicago.csv
* new_york_city.csv
* washington.csv
